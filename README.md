La Momia Ataca


 Descripción:
Este proyecto simula un sistema donde el jugador actúa como un Arquitecto de la Supervivencia, gestionando recursos para mantener con vida a exploradores dentro de una pirámide.

 Objetivo:
Sobrevivir el mayor tiempo posible mientras una momia se mueve y amenaza a los exploradores.

Funcionalidades:
Movimiento de la momia
Gestión de energía
Detección de colisiones
Simulación por turnos

Clases principales:
Main → Ejecuta el programa
Juego → Controla la lógica
Explorador → Personajes
Momia → Enemigo

Tecnologías:
Java
Programación Concurrente


Ejecución:

Compilar:
javac *.java

Ejecutar:
java Main
